## Features

- Light/dark mode toggle
- Fully Responsive
- Realtime Forecast
- Shows humidity, UV Index, Air quality, etc...


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
